# AI Agent Guidelines

## Project Overview

This is a personal blog built with React 18 and Create React App. It uses Mantine for UI components, React Router for navigation, and is deployed to GitHub Pages.

### Key Technologies
- React 18
- Mantine UI (v7)
- React Router DOM v6
- React Latex (for mathematical content)
- React Syntax Highlighter (for code blocks)

### Project Structure
- `src/pages/` - Page components (Home, AboutMe, blog posts)
- `src/App.js` - Main app with routing and navigation
- `public/` - Static assets

## Rules

1. **Python virtual environment**: Before running any Python command, activate the virtual environment:
   ```bash
   source venv/bin/activate
   ```

2. **Dependencies**: Use `npm install` for JavaScript dependencies, `pip install` within the activated venv for Python.

3. **Development server**: Run `npm start` to start the local dev server.

4. **Build**: Run `npm run build` to create a production build.

5. **Deployment**: Run `npm run deploy` to deploy to GitHub Pages.
